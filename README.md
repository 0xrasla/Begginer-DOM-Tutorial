# DOM Manipulation — A Step-by-Step Beginner Tutorial

Welcome! This tutorial teaches you **DOM Manipulation** from scratch, one step at a time.

---

## What is DOM Manipulation?

**DOM Manipulation** means using JavaScript to change the content, structure, or appearance of a web page after it has loaded in the browser.

Without DOM manipulation, a web page is just a static document. With it, you can:

- Change text and images dynamically
- Show/hide content based on user actions
- Build interactive forms, menus, and games
- Create entire applications (like Gmail or Google Maps)

## Prerequisites

- Basic HTML (tags, attributes, nesting)
- Basic JavaScript (variables, functions, `let`/`const`)
- A text editor (VS Code recommended)
- A web browser (Chrome or Firefox recommended)

You do **not** need any frameworks. Everything here uses plain vanilla JavaScript.

---

## How to Use This Tutorial

1. Go through lessons **in order** — each builds on the previous one
2. **Type the code yourself** — don't just read it
3. Open the browser console (`F12` or `Cmd+Option+J`) to test examples
4. Try modifying the examples to see what happens
5. Build the mini-project at the end to solidify your skills

---

## Quick Start

Open your browser's developer console and type:

```javascript
document.title = "My First DOM Change!";
```

You just manipulated the DOM! Now let's learn how to do much more.
