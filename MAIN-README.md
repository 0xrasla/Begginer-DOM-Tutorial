## About This Course

Welcome to the **DOM Manipulation Course** — a complete, beginner-friendly tutorial designed to take you from knowing nothing about the DOM to building real interactive web applications.

This course is created as part of the **My Coding Diary** series on YouTube, where I document my coding journey and break down complex topics into simple, easy-to-understand lessons.

---

## Who Is This For?

This course is perfect for:

| Audience | Why This Course Helps |
|---|---|
| **Complete Beginners** | No DOM knowledge needed — we start from the very basics |
| **HTML/CSS Learners** | You know how to build static pages — now learn to make them interactive |
| **New JavaScript Developers** | DOM manipulation is the foundation of front-end development |
| **Self-Taught Programmers** | Fill in the gaps with structured, step-by-step lessons |
| **Students & Bootcampers** | A supplementary resource to reinforce what you're learning |
| **Career Switchers** | Breaking into web development? This is where to start |

### Prerequisites

- Basic HTML (tags, attributes, nesting)
- Basic JavaScript (variables, functions, `let`/`const`)
- A text editor (VS Code recommended)
- A web browser (Chrome or Firefox recommended)
- Curiosity and willingness to practice

> **You do NOT need any frameworks.** This course uses plain vanilla JavaScript — the same foundation that React, Vue, and Angular are built on.

---

## What You'll Learn

By the end of this course, you will be able to:

- [x] Understand how the browser builds the DOM from HTML
- [x] Select any element on a page using multiple methods
- [x] Change text, HTML content, and styles dynamically
- [x] Work with attributes and custom `data-*` properties
- [x] Create, add, clone, and remove elements
- [x] Handle user interactions (clicks, key presses, forms)
- [x] Navigate the DOM tree (parents, children, siblings)
- [x] Build a complete To-Do List application from scratch
- [x] Follow best practices and avoid common mistakes

## Course Structure

The course is divided into **11 lessons**, each building on the previous one.

| # | Lesson | Description | Duration |
|---|---|---|---|
| 0 | [Introduction](./README.md) | Course overview and setup | 5 min |
| 1 | [The DOM Tree](./01-the-dom-tree/) | How the browser turns HTML into a tree | 10 min |
| 2 | [Selecting Elements](./02-selecting-elements/) | Finding elements with querySelector, getElementById | 15 min |
| 3 | [Changing Content](./03-changing-content/) | Modifying text and HTML (textContent, innerHTML) | 10 min |
| 4 | [Changing Styles](./04-changing-styles/) | Inline styles and CSS classes (classList) | 15 min |
| 5 | [Changing Attributes](./05-changing-attributes/) | Working with src, href, data-* attributes | 15 min |
| 6 | [Creating & Removing Elements](./06-creating-removing-elements/) | Dynamic DOM manipulation | 15 min |
| 7 | [Handling Events](./07-handling-events/) | Clicks, forms, keyboard, event delegation | 20 min |
| 8 | [Traversing the DOM](./08-traversing-the-dom/) | Navigating parents, children, siblings | 15 min |
| 9 | [Mini Project: To-Do List](./09-mini-project/) | Build a real app with everything learned | 20 min |
| 10 | [Best Practices](./10-best-practices/) | Common mistakes and how to avoid them | 10 min |
| 11 | [Cheat Sheet](./11-cheat-sheet/) | Quick reference for all methods | 5 min |

> **Total estimated time: ~2.5 hours** (including practice)

## How to Use This Course

### Option 1: Follow Along with YouTube

Watch the **My Coding Diary** series on YouTube and use these notes as supplementary material.

### Option 2: Self-Paced Reading

1. Start with the [Introduction](./README.md)
2. Go through each lesson **in order** — each builds on the previous
3. **Type the code yourself** — don't just read it
4. Open the browser console (`F12` or `Cmd+Option+J`) to test examples
5. Try modifying the examples to see what happens
6. Build the mini-project at the end to solidify your skills

## Quick Start

Clone this repository and open the introduction:

```bash
git clone https://github.com/0xRasla/DOM-Manipulation-Course.git
cd DOM-Manipulation-Course
```

Or just open any `README.md` file in your favorite Markdown viewer (VS Code has a built-in one — right-click and select "Open Preview").

To run the mini-project:

```bash
cd 09-mini-project
open index.html    # macOS
# or just double-click index.html in your file explorer
```
